# 梦幻彩虹蛇游戏

这是一个精美的HTML5彩虹贪吃蛇游戏，具有炫彩的视觉效果和流畅的游戏体验。无需安装任何插件，只需一个浏览器即可享受经典游戏的乐趣！

## 游戏特色

- 🌈 绚丽的彩虹蛇身，随着长度增加呈现渐变色彩
- ✨ 梦幻的星空背景和粒子特效
- 🍎 多种食物类型，每次吃到食物都会产生精美的粒子爆炸效果
- 🎮 经典游戏模式，撞墙游戏结束，考验玩家操作技巧
- 📱 完美支持移动设备，触摸控制和屏幕自适应

## 如何游玩

1. 在电脑上：
   - 使用方向键控制蛇的移动方向
   - 避免撞到墙壁和自己的身体
   - 收集尽可能多的食物让彩虹蛇变得更长

2. 在手机上：
   - 使用屏幕上的虚拟方向键或在游戏区域滑动控制方向
   - 竖屏玩游戏以获得最佳体验
   - 触摸控制灵敏且直观

## 技术特点

- 纯HTML5/CSS3/JavaScript实现，无需任何外部库
- 响应式设计，适配各种屏幕尺寸
- 平滑的动画效果和精确的碰撞检测
- 针对移动设备优化的触摸控制系统
- 详细的游戏日志记录，方便调试和性能优化

## 安装与运行

1. 克隆或下载本仓库
2. 直接在浏览器中打开tanchishe.html文件
3. 开始游戏冒险！

或者将文件部署到任何Web服务器，通过URL访问。

## 自定义开发

如果你想对游戏进行修改或扩展，代码结构清晰，易于理解：

- HTML部分定义了游戏界面元素
- CSS样式提供了美观的视觉效果
- JavaScript实现了所有游戏逻辑

你可以轻松调整食物类型、游戏速度、蛇的颜色等参数。
